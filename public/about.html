About

NextMetro

NextMetro is a real-time tracker for the Washington D.C. Metrorail system. It pulls live data from the WMATA API to display train arrivals, system status, service alerts, station facility conditions, and fare information — all in one place.

The interface draws from the Metro's own visual language: the concrete vault ceilings, pylon signage, and PIDS (Passenger Information Display System) boards found in every station. Built as a personal project by Nick Poole.

Features

  • Real-time train arrival board with PIDS-style display
  • Searchable station selector covering all 98 WMATA stations
  • Multi-platform station support (Metro Center, Gallery Place, L'Enfant Plaza, Fort Totten)
  • Live system status ticker and sidebar for all six Metro lines
  • Station-specific service alerts
  • Elevator and escalator outage tracking
  • Fare calculator with peak, off-peak, and senior pricing
  • Mobile-responsive, accessibility-minded design

Data & Accuracy

Train arrivals refresh every 25 seconds. System incidents update every 60 seconds. Elevator and escalator status updates every 2 minutes. All data comes directly from the WMATA API — accuracy depends on WMATA's real-time feed.

NextMetro is an independent project and is not affiliated with, endorsed by, or connected to the Washington Metropolitan Area Transit Authority (WMATA). All transit data is sourced from the WMATA Developer API.
